# Task Manager Application

## Features
- User Login & Register
- Task Management (Create, Update, Delete)
- Task Status Tracking (TODO, IN_PROGRESS, DONE)
- Role-based access (Admin / User)
- Filters (Status & User)

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

## Run Locally

### Backend
Run Spring Boot application

### Frontend
cd frontend
npm install
npm run dev

## URLs

Backend: http://localhost:8080
